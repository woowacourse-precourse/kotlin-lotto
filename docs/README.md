## 🚀 기능 목록

### Application

- [x] 로또 게임을 시작한다.
- [x] 통계를 출력한다.

### Lotto
- [x] 로또 번호를 알려준다.

### Machine

- [x] 금액에 따라 로또의 개수를 받는다.
    - [x] 예외) 금액이 1000으로 나누어 떨어지지 않으면 ERROR 출력

### Ticket

- [x] 로또를 위한 번호 6개를 생성한다.
    - [x] 로또번호 숫자 범위는 1 ~ 45까지이다.
    - [x] 생성된 로또 번호를 오름차순으로 정렬한다.

### Broadcast

- [x] 당첨된 통계를 계산한다.
- [x] 수익률을 계산한다.
