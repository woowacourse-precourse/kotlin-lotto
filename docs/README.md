_**기능별목록**_
 
[]로또 구매 금액 입력  
[]당첨 번호 입력  
[]보너스 번호 입력  
[]구매 금액 만큼 랜덤 숫자 6개 뽑기
[]발행한 로또 수량 및 번호 출력(오름차순으로)  
[]당첨 번호가 몇 등에 당첨되는지 계산
[]당첨 내역 출력  
[]수익률 계산(둘째 자리에서 반올림)  
[]수익률 출력  


**_예외사항_**

-숫자가 1~45의 범위를 넘어간 경우  
-입력시 당첨 번호를 6개에 맞지 않게 입력한 경우
-입력시 보너스 번호를 1개에 맞지 않게 입력한 경우
-입력 금액이 1000원 단위가 아닌 경우
