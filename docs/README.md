# **💵 로또 게임 기능 요구 사항**

## **Goal : 사용자가 구매한 로또 번호와 당첨 번호를 비교하여 당첨 내역 및 수익률을 출력**
---

## 💡 **기능 구현 목록**

### **1. 구입 금액을 입력받는 기능**

- 로또 1장의 가격은 1,000원
- 구입 금액은 1,000원 단위로 입력

### **2. 구입 금액에 해당하는 로또의 개수를 출력하는 기능**

- ex) 8개를 구매했습니다.

### **3. 구입한 개수만큼 로또를 발행하는 기능**

- 1개의 로또를 발행할 때마다 중복되지 않는 6개의 숫자를 랜덤으로 뽑음

### **4. 발행한 각 로또들을 출력하는 기능**

- 각 로또 번호는 오름차순으로 정렬하여 출력
- 배열의 형식으로 출력 ex) [8, 21, 23, 41, 42, 43]

### **5. 당첨 번호를 입력받는 기능**

- 쉼표를 기준으로 6개의 숫자를 입력

### **6. 보너스 번호를 입력받는 기능**

- 1개의 숫자를 입력

### **7. 각 로또 별 당첨 여부를 계산하는 기능**

### **8. 당첨 내역을 출력하는 기능**

- ex)<br>
  3개 일치 (5,000원) - 1개<br>
  4개 일치 (50,000원) - 0개<br>
  5개 일치 (1,500,000원) - 0개<br>
  5개 일치, 보너스 볼 일치 (30,000,000원) - 0개<br>
  6개 일치 (2,000,000,000원) - 0개<br>

### **9. 총 수익률을 계산하는 기능**

- 수익률은 소수점 둘째 자리에서 반올림

### **10. 총 수익률을 출력하는 기능**

- ex) 총 수익률은 62.5%입니다.
