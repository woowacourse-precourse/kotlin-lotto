## 🚀 기능 목록

### Lotto

- [ ] 로또 게임을 시작한다.
  - [ ] 금액에 따라 로또의 개수를 받는다.
    - [ ] 예외) 금액이 1000으로 나누어 떨어지지 않으면 ERROR 출력
  - [ ] 당첨 번호를 입력한다.
  - [ ] 보너스 번호를 입력한다.
  - [ ] 생성된 번호와 로또 번호가 맞는지 확인한다.

### Machine  

- [ ] 로또를 위한 번호 6개를 생성한다.
  - [ ] 로또번호 숫자 범위는 1 ~ 45까지이다.
  - [ ] 중복되지 않는 숫자 6개 + 1개이다.
  - [ ] 생성된 로또 번호를 오름차순으로 정렬해 출력한다.

### Broadcast
  - [ ] 당첨된 통계를 발표한다.
  - [ ] 수익률을 출력한다.
