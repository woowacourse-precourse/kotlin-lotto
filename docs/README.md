## 기능 명세서

### View
* 구입 금액을 입력받음
* 발행한 로또 수량 및 번호를 출력
* 당첨 번호 입력 받음
* 보너스 번호 입력 받음
* 당첨 내역을 출력
* 수익률 출력

### Logic
* 로또 번호 발행
* 당첨 계산
    * 동일한 번호의 갯수 세기
    * 보너스 번호를 맞췄는지 검사
    * 등수 파악 후 Map 업데이트
* 수익률 계산

### Exception
* 입력 금액이 1000으로 나누어 떨어지지 않을 때 예외
* 로또 번호가 1~45의 정수가 아닐 때 예외
* 보너스 번호가 1~45의 정수가 아닐 때 예외
* 보너스 번호가 당첨 번호와 중복될 때 예외
