> * 금액 입력받는 함수
> * x개 구매 -> (랜덤으로 서로다른숫자 6개) * x번 발행하는 함수
> * 당첨번호 입력받는 함수
> * 보너스 번호 입력받는 함수 -> 에러 함수 호출
> * 당첨 수 계산 함수
> * 수익률 계산 함수

> * 에러 함수

## Class layout
The contents of a class should go in the following order:
>
> Property declarations and initializer blocks
  
  Secondary constructors
  
  Method declarations
  
  Companion object
